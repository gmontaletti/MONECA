# Setup Parallel Processing for Auto-Tuning

Configures parallel processing cluster for enhanced performance.

## Usage

``` r
setup_parallel_processing(n_cores = NULL, verbose = FALSE)
```

## Arguments

- n_cores:

  Number of cores to use. If NULL, auto-detects optimal number.

- verbose:

  Logical indicating whether to print setup messages.

## Value

List containing parallel setup information.
