# Print Method for moneca_model_comparison Objects

Displays a formatted comparison of multiple log-linear model fits,
including a fit statistics table and likelihood-ratio tests for nested
models.

## Usage

``` r
# S3 method for class 'moneca_model_comparison'
print(x, digits = 3, ...)
```

## Arguments

- x:

  A `moneca_model_comparison` object.

- digits:

  Integer specifying the number of significant digits for numeric
  output. Default is 3.

- ...:

  Additional arguments (ignored).

## Value

Invisibly returns the object.
