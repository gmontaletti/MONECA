# Print Method for moneca_loglinear Objects

Displays a formatted summary of a log-linear model fit for a mobility
table, including deviance, degrees of freedom, p-value, and fit indices.

## Usage

``` r
# S3 method for class 'moneca_loglinear'
print(x, digits = 3, ...)
```

## Arguments

- x:

  A `moneca_loglinear` object.

- digits:

  Integer specifying the number of significant digits for numeric
  output. Default is 3.

- ...:

  Additional arguments (ignored).

## Value

Invisibly returns the object.
