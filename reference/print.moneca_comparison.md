# Print Method for moneca_comparison Objects

Displays a structured summary of the comparison between two MONECA
clustering results.

## Usage

``` r
# S3 method for class 'moneca_comparison'
print(x, ...)
```

## Arguments

- x:

  A \`moneca_comparison\` object returned by
  \[compare_moneca_results()\].

- ...:

  Additional arguments (currently unused).

## Value

Invisibly returns \`x\`.
